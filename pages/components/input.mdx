# Input

인봄에서 사용자에게 정보를 입력받는 모든 곳에서 사용하는 컴포넌트 모음이에요.
회원가입, 견적 신청, 업체 검색까지 — 폼이 들어가는 거의 모든 페이지에서 쓰였어요.
각각의 입력 상황에 맞게 특화된 컴포넌트로 분리해서 관리했어요.

---

## Input

가장 기본이 되는 텍스트 입력 필드예요.

`React.InputHTMLAttributes`를 extends해서
`value`, `onChange`, `readOnly`, `maxLength` 같은
네이티브 input 속성을 별도로 선언하지 않아도 모두 그대로 사용할 수 있어요.
`forwardRef`로 감싸서 부모에서 ref를 직접 제어할 수 있도록 했어요.

<Input placeholder="이름을 입력해주세요" />
```tsx
<Input placeholder="이름을 입력해주세요" />

// readOnly

<Input value="변경 불가 텍스트" readOnly />

// 비활성화

<Input placeholder="비활성화 상태" disabled />
```

## PasswordInput

비밀번호 입력에 특화된 컴포넌트예요.

눈 모양 아이콘을 클릭하면 비밀번호가 보이고 숨겨지는 토글 기능이 있어요.
`showPassword` 상태를 내부에서 관리하고, `type`을 `text`와 `password` 사이에서 전환해요.
토글 버튼에는 `aria-label`을 적용해서 스크린 리더 사용자도 현재 상태를 알 수 있도록 했어요.

<PasswordInput placeholder="비밀번호를 입력하세요" />
```tsx
<PasswordInput placeholder="비밀번호를 입력하세요" />
```

---

## SearchInput

검색어 입력에 특화된 컴포넌트예요.

input 왼쪽에 검색 아이콘이 고정으로 붙어있어요.
아이콘을 input 내부가 아닌 wrapper div로 감싸서 배치한 건,
CSS만으로 아이콘 위치를 안정적으로 잡기 위해서였어요.
`forwardRef`를 적용해서 검색 버튼이나 폼에서 ref로 포커스 제어도 가능해요.

<SearchInput placeholder="업체명, 지역을 검색해보세요" />
```tsx
<SearchInput placeholder="업체명, 지역을 검색해보세요" />
```

## Select

드롭다운 선택 컴포넌트예요.

`options` 배열을 받아서 `<option>` 목록을 렌더링해요.
빈 값이 선택됐을 때는 `null`을 반환하도록 처리했어요.
실제로 "선택 안 함"과 "빈 문자열"을 구분해야 하는 경우가 있었거든요.
커스텀 화살표 아이콘을 wrapper로 덧씌워서 브라우저 기본 스타일을 통일했어요.

<Select
  options={[
    { label: '서울', value: 'seoul' },
    { label: '경기', value: 'gyeonggi' },
    { label: '인천', value: 'incheon' },
  ]}
  placeholder="지역을 선택하세요"
/>
```tsx
<Select
  options={[
    { label: '서울', value: 'seoul' },
    { label: '경기', value: 'gyeonggi' },
    { label: '인천', value: 'incheon' },
  ]}
  placeholder="지역을 선택하세요"
  onChange={(value) => console.log(value)}
/>
```

## DatePicker

날짜 범위를 선택하는 컴포넌트예요.

인봄에서 견적 신청 시 시공 가능 기간을 입력받는 용도로 만들었어요.
시작일을 먼저 클릭하고, 종료일을 클릭하면 범위가 확정돼요.
마우스를 올리면 선택될 범위를 미리 보여주는 hover 프리뷰도 구현했어요.

오늘 이전 날짜는 선택할 수 없도록 막았어요.
과거 날짜로 견적을 신청하는 건 의미가 없었거든요.
종료일까지 선택이 완료되면 달력이 자동으로 닫혀요.
외부 영역 클릭 시에도 닫히도록 `useEffect`로 이벤트 리스너를 등록했어요.

<DatePicker placeholder="시공 가능 기간을 선택해주세요" />
```tsx
<DatePicker
  placeholder="시공 가능 기간을 선택해주세요"
  onChange={({ startDate, endDate }) => {
    console.log('시작일:', startDate);
    console.log('종료일:', endDate);
  }}
/>
```

## TimePicker

시간대를 선택하는 컴포넌트예요.

인봄에서 견적 신청 시 방문 가능한 시간대를 선택받는 용도로 사용했어요.
'아무때나 좋아요'를 선택하면 다른 슬롯이 모두 해제되고,
반대로 특정 시간대를 선택하면 '아무때나'가 자동으로 해제돼요.
두 상태가 동시에 선택되는 게 논리적으로 맞지 않았거든요.

<TimePicker onChange={(slots) => console.log(slots)} />
```tsx
<TimePicker
  value={['10-12', '13-15']}
  onChange={(slots) => console.log(slots)}
/>
```

## Props

### Input

| Prop          | Type                  | Default | Description              |
| ------------- | --------------------- | ------- | ------------------------ |
| `type`        | `string`              | `text`  | input 타입               |
| `placeholder` | `string`              | -       | 플레이스홀더             |
| `className`   | `string`              | -       | 추가 클래스명            |
| `...props`    | `InputHTMLAttributes` | -       | 네이티브 input 속성 전체 |

### PasswordInput

| Prop          | Type                  | Default                   | Description              |
| ------------- | --------------------- | ------------------------- | ------------------------ |
| `placeholder` | `string`              | `'비밀번호를 입력하세요'` | 플레이스홀더             |
| `className`   | `string`              | -                         | 추가 클래스명            |
| `...props`    | `InputHTMLAttributes` | -                         | 네이티브 input 속성 전체 |

### SearchInput

| Prop          | Type                  | Default                 | Description              |
| ------------- | --------------------- | ----------------------- | ------------------------ |
| `placeholder` | `string`              | `'검색어를 입력하세요'` | 플레이스홀더             |
| `className`   | `string`              | -                       | 추가 클래스명            |
| `...props`    | `InputHTMLAttributes` | -                       | 네이티브 input 속성 전체 |

### Select

| Prop          | Type                                 | Default        | Description   |
| ------------- | ------------------------------------ | -------------- | ------------- |
| `options`     | `{ label: string; value: string }[]` | `[]`           | 옵션 목록     |
| `placeholder` | `string`                             | `'선택하세요'` | 플레이스홀더  |
| `value`       | `string \| null`                     | -              | 선택된 값     |
| `onChange`    | `(value: string \| null) => void`    | -              | 변경 핸들러   |
| `className`   | `string`                             | -              | 추가 클래스명 |

### DatePicker

| Prop          | Type                         | Default                 | Description       |
| ------------- | ---------------------------- | ----------------------- | ----------------- |
| `placeholder` | `string`                     | `'날짜를 선택해주세요'` | 플레이스홀더      |

| `onChange`    | `(range: DateRange) => void` | -                       | 범위 확정 시 콜백 |
| `className`   | `string`                     | -                       | 추가 클래스명     |

### TimePicker

| Prop       | Type                        | Default | Description    |
| ---------- | --------------------------- | ------- | -------------- |
| `value`    | `string[]`                  | -       | 초기 선택 슬롯 |
| `onChange` | `(slots: string[]) => void` | -       | 변경 핸들러    |
